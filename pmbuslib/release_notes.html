Introduction

This is the MSP430™ PMBus Library.

The MSP430™ PMBus Library is a royalty-free set of API's that provide support for master protocols in a PMBus communication system.

      Examples

      The examples may be built with either IAR or CCS. To build the examples navigate to example folder, select a project and click the .eww project files for IAR or import the .projectspec file for CCS. See the PMBus Users Guide for detailed usage instructions.

      Current supported experimenters boards are found in the Device Support section of the release notes.
