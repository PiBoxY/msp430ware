Field Documentation

const msp_biquad_df2_q15_coeffs* msp_biquad_df2_q15_params::coeffs

Pointer to DF2 filter coefficients. This data block must be allocated in shared RAM when using LEA.

uint16_t msp_biquad_df2_q15_params::length

Length of the source data, must be a multiple of two.

msp_biquad_df2_q15_states* msp_biquad_df2_q15_params::states

Pointer to an array of length two used to store the state of the operation. When continuous operation is desired the previous state needs to be passed to the next biquad operation. This data block must be allocated in shared RAM when using LEA.
