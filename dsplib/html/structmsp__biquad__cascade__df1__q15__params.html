const msp_biquad_df1_q15_coeffs* msp_biquad_cascade_df1_q15_params::coeffs

Pointer to an array of DF1 filter coefficients of length stages. This data block must be allocated in shared RAM when using LEA.

uint16_t msp_biquad_cascade_df1_q15_params::stages

Number of cascaded biquad filters, typically the filter order divided by two.

msp_biquad_df1_q15_states* msp_biquad_cascade_df1_q15_params::states

Pointer to an array of DF1 filter states of length stages. When continuous operation is desired the previous states must be passed to the next cascaded biquad operation. This data block must be allocated in shared RAM when using LEA.

Examples:
filter_ex3_biquad_cascade_df1_q15.c.
