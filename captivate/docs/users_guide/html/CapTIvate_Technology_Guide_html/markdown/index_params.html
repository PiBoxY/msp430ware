Captivate Parameter Index

Captivate systems are configured through modification of parameters. This index provides links to detailed descriptions of each paramter. There are three main parameter types: Conversion Control Parameters, Tuning Parameters, and Controller Parameters.

Sensor Conversion Control Parameters
Conversion control parameters specify the details of how the Captivate peripheral is configured when measuring a specific sensor. Each sensor has its own set of conversion control parameters.

1. Conversion Count
2. Conversion Gain
3. Conversion Frequency Divider
4. Conversion Phase Lengths
5. Modulation
6. Pin Idle State
7. Sample Capacitor Discharge Level
8. Bias Current (Mutual Capacitance Only)
9. Sync Parameters
11. Auto Power-Down
12. Gain (Coarse and Fine)
13. Offset Subtraction

Sensor Tuning Parameters
Tuning parameters specify the details of how the Captivate software library processes the measurements it gets back from the peripheral to turn them into useful information. Each sensor has its own set of conversion control parameters.

1. Proximity Thresholds
2. Touch Thresholds
3. De-Bounce
4. Sensor Timeout Threshold
5. Runtime Re-Calibration
6. Count Filter
7. Long Term Average (LTA) Filter
8. Error Threshold
9. Negative Touch Threshold
10. Position Filter
11. Slider/Wheel Desired Resolution
12. Slider Trim

Controller Parameters (Global System Parameters)
Controller parameters specify global system parameters that apply to all sensors.

1. Element Data Transmit Enable
2. Sensor Data Transmit Enable
3. System Report Rate
