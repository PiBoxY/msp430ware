Tap Time Minimum Threshold

This parameter sets the minimum time (#samples) required for a valid tap gesture. If the number of samples is less than this parameter, the gesture is ignored. A valid tap requires the number of samples to fall with the min and max parameters.

Tap Time Maximum Threshold

This parameter sets the maximum time (#samples) for a valid tap gesture . If the number of samples is greater than this parameter, the gesture is ignored. A valid tap requires the number of samples to fall with the min and max parameters.

Range of Valid Threshold Parameters
The Tap Time Minimum threshold may be set from 1 to 65535 and should be set less than Tap Time Maximum.

The Tap Time Maximum threshold may be set from 1 to 65535 and should be set greater than Tap Time Minimum.

Affected Software Parameters
The Duration Threshold parameter corresponds to the
ui16TapTimeMin
member of the
tTrackPadSensorParams type in the CapTIvate Touch Library.

The Swipe Distance Threshold parameter corresponds to the
ui16TapTimeMax
member of the
tTrackPadSensorParams type in the CapTIvate Touch Library.



Double Tap Time Maximum Threshold

This parameter sets the maximum time (#samples) for a valid double tap gesture. If the number of samples is greater than this parameter, the gesture is ignored. A valid double tap requires the number of samples to be less than the max parameter.

Range of Valid Threshold Parameters
The Double Tap Time Maximum threshold may be set from 1 to 65535.

Affected Software Parameters
The Double Tap Time Max threshold parameter corresponds to the
ui16DoubleTapTimeMax
member of the
tTrackPadSensorParams type in the CapTIvate Touch Library.

Range of Valid Threshold Parameters
The Tap and Hold Time Minimum threshold may be set from 1 to 65535.

Affected Software Parameters

member of the
