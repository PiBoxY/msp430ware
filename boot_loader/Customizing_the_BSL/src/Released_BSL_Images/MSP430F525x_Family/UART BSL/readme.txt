BSL for MSP430F525x family
==========================

BSL v00.08.08.04 uses pins in the DVCC supply domain. (Factory programmed)
 - TXD P6.1
 - RXD P6.2

BSL v00.08.08.53 uses pins in the DVIO supply domain.
 - TXD P3.3
 - RXD P3.4

Please refer to the device's datasheet for more information.
