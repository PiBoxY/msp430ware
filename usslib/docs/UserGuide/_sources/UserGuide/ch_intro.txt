Introduction
============

Sections
--------

The Ultrasonic Sensing Design Center (USSDC) front-end is split into seven main sections:

-  `Configuration <ch_config.html>`__
-  `Waveform Plots <ch_plotting.html>`__
-  `ADC Capture Functionality <ch_adc.html>`__
-  `Frequency Sweep <ch_freqsweep.html>`__
-  `Calibration <ch_calibration.html>`__
-  `Debug Waveform <ch_debug_graph.html>`__
-  `Error Handling <ch_errors.html>`__

Logging information and system requirements are explained here:

Logging
~~~~~~~

The `Logging <ch_logging.html>`__ chapter explains the logging capabilities supported by USSDC.

System Requirements
~~~~~~~~~~~~~~~~~~~

The `System Requirements <ch_sysreqs.html>`__ chapter includes the Operating System and Java requirements.

Communication Protocol Specification
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The `Communication Protocol Spec <ch_protocol.html>`__ chapter introduces the command details and protocol used between USSDC and the target.

Revision History
~~~~~~~~~~~~~~~~

The `Revision History <ch_revision.html>`__ chapter tracks changes in the Ultrasonic Design Center and Ultrasonic Software Library.

Navigation
----------

The sections are accessed via the navigation menu below the main menu bar. Click the corresponding tab to access its contents.

Getting Started
---------------

The GUI requires connection to a target board via USB. To connect or disconnect to a device, either use the Communications tab at the top (Alt + C), then click Connect or Disconnect. Alternatively, use F1 to connect and F2 to disconnect.

The current connection status and target at the bottom left corner of the screen.

   :alt: HID Connection Bar
