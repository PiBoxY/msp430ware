Getting Started
---------------

USS Design Center comes with a Quick Start guide for the MSP430FR6047. The Quick Start guide can be accessed `here <http://www.ti.com/lit/pdf/slau720>`__, or directly from the GUI via the Help item on the main menu bar (Alt + H).

.. figure:: images/quick_start_guide.png
   :alt: Opening the Quick Start guide

   Opening the Quick Start guide

The GUI requires connection to a target board via USB. To connect or disconnect to a device, either use the Communications tab at the top (Alt + C), then click Connect or Disconnect. Alternatively, use F1 to connect and F2 to disconnect.

The current connection status and target at the bottom left corner of the screen.

.. figure:: images/hid_conn_bar.png
   :alt: HID Connection Bar

   HID Connection Bar
