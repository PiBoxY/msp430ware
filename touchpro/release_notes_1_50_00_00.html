Introduction

The TouchPro tool is a PC based tool to aid in the design and tuning of capacitive touch button(s), slider(s) and wheels(s).  The tool receives and displays captouch sensor data to help the user quickly and easily evaluate, diagnose and tune button, slider and wheel designs.

Features of the Touch Pro Tool include:
• Receive real-time data from a target board, including up to ten signals
• Points connected or bar chart display styles for viewing real-time data waveforms
• User configurable automatic ranging or manual ranging of the y-axis
• Data recording and recall
• Print waveforms
• User selectable individual signal or multiple signal waveform display

Installation and Usage

Run the installer and it will create TouchPro product and shortcuts under Texas Instruments.

See the Users Guide for platform-specific instructions.

The TouchPro  installed directory contains the following components.

Tool Executable An executable jar file named touchPro.jar and a batch file named touchPro.bat
documents Quick Start User guide to get started with the tool
Com port driver Included in the installed directory is the RXTXSerial.dll 
examples CCS and IAR project examples that transmit data from target to host using big banging and UART interface
Library Included in the example projects are the CTS library files
Source and header files Included in the example projects are application specific .c and .h source and header files
Java Runtime EnvironmentJRE6 evnvironment included

Examples

The TouchPro tool's installed directory contains example CCS and IAR project files to use with the TouchPro tool.  Both code examples were designed to work with the MSP430G2553 microcontroller. The example labeled TouchProTool_Demo_Bit_Banging supports the MSP430G2xx2 and MSP430G2xx3 Value Line MSP microcontrollers. The example that is labeled TouchProTool_Demo_UART supports only MSP430G2xx3 Value Line MSP microcontrollers, because it requires an UART interface. The UART module should be wired according to the instructions in the LaunchPad documentation.
