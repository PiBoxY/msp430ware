Change Log

This chapter describes the differences between releases of the Energy Measurement software library. The latest (most current) builds are shown first.

1.30.00

Build Date: June 2018

This is a feature addition build.

Features Additions from Previous Version

  • Added new API EM_delayCurrent90Degrees(EM_Metrology *metro,EM_Phase_Configuration *phase,EM_Phase_BG_Results *bgData) to support Rogowski Coil
  • Added new variables "rogowskiPrevI90" and "iPrevSample" to EM_Phase_BG_Results structure to keep track of the Rogowski Coil current
  • Added support for IAR Embedded Workbench 7.10 or higher
  • Switched Frequency Locked Loop (FLL) reference clock from REFO to XT1 for MSP430F6xx

Functional Changes from Previous Version

  • None

1.20.00 (Initial Release to Market)

Build Date: April 2018

This is the initial release to market build of the library to support MSP430i2xx and MSP430F6xx devices.

Features Additions from Previous Version

None, this is the first release

Functional Changes from Previous Version

None, this is the first release
