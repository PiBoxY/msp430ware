Welcome to the MSP-EXP430FR5994 LaunchPad!
********************************************************************************

In this package, you will find software examples and source codes that can help
you quickly get started on developing your own project.

The software example projects are located in the 'Source' folder:
 1. BlinkLED_MSP430FR5994        | Blink LED example source code
 2. OutOfBox_MSP430FR5994        | The out of box demo source code
 3. BOOSTXL-AUDIO_RecordPlayback_MSP430FR5994 | BOOSTXL-AUDIO Record/Playback example
 4. BOOSTXL-SENSORS_SensorGUI_MSP430FR5994 | BOOSTXL-SENSORS GUI Example
 5. BOOSTXL-BATPAKMKII_FuelGauge_MSP430FR5994 | Battery Boosterpack demo
 6. BOOSTXL-AUDIO_430BOOST-SHARP_FFT_FIR_LEA_MSP430FR5994 | FFT, Filtering LEA demo

In addition, prebuilt TI-TXT images are located in the 'Binary' folder. These
images are ready to be downloaded onto your LaunchPad using a free MSP430
programming interface, such as the included MSP430Flasher. Just run the
"Program <Example>" script to download the corresponding project to your LaunchPad!

The Out of Box Demo GUI can be launched by running the executable inside the
'GUI/Binary' folder. Source for the GUI can also be found in a zip within the
'GUI/Source' folder.

You can also find various web links in the 'Documentation' folder pointing to
the Tool Page, Quick Start Guide, and User's Guide

For more resources, be sure to visit www.ti.com/tool/msp-exp430FR5994
