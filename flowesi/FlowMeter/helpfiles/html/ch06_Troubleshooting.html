Troubleshooting

"Start Calibration" hangs

If Start Calibration hanged, follow these steps:

  1. Click Cancel at the bottom of Target Comm Panel.
  2. Decrease distance between rotor and LC sensors.
  3. Connect to the HID/UART bridge.
  4. Click "Start Calibration" Button.

FlowESI GUI crashed

Report this problem to msp43.nosp@m.0war.nosp@m.e@lis.nosp@m.t.ti.nosp@m..com. Make sure to attach the following file in the email:

  • USERPROFILE%/DesignCenter/DesigCenter.log
  • USERPROFILE%/DesignCenter/DesigCenter.dat
  • USERPROFILE%/DesignCenter/<PROJECT_NAME>/DesignCenter.ser

Typically, USERPROFILE% points to "C:\Users\{username}"

Cannot open HID port on linux

If you are not running the application using a root account, you may not have permissions to the HID device mounted by the MSP microcontroller.

To enable access perform the one-time setup step:

  • Create the file /etc/udev/rules.d/ti_hid.rules.
  • Add the lines below to the file and save the file.
1 # Allow open access to any usb device with the default Texas Instruments MSP
2 # Vendor Id of 0x2047
3 ATTRS{idVendor}=="2047", MODE="0666"
