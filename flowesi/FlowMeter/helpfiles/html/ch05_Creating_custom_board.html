Creating a Custom Board Configuration File

Creating a Custom Board File

Creating a custom board file requires editing the file under "FlowMeter/config/ToolConfig/Boards/CUSTOM_BOARD.xml"

Enabling Rotation LED

To specify that your custom board has the option toggle the rotation LED update the "<rotationLED>" tag.

Change: 

<rotationLed available="false">

    <GPIOPort></GPIOPort>

    <GPIOPin></GPIOPin>
</rotationLed>

To: 

<rotationLed available="true">

    <GPIOPort>GPIO_PORT_PX</GPIOPort>

    <GPIOPin>GPIO_PINX</GPIOPin>
</rotationLed>

IMPORTANT: Make sure you replace GPIO_PORT_PX and GPIO_PINX for the appropriate Port and Pin number in you application.

<reCalLed available="false">

Enabling Rotation LED

If you custom board has the same LCD as the EVM430-FR6989 (Please refer to MSP430FR6989 EVM documentation). You can enable the LCD functions by updating "<LCD>" tag. Change: 

<LCD available="false"/>

To: 

<LCD available="true"/>

Enabling Rotation LED

If you custom board has the same HID/UART bridge as the EVM430-FR6989 (Please refer to MSP430FR6989 EVM documentation). You can enable the Target Communication functions by updating "<targetCOMM>" tag.

Change: 

<targetCOMM available="false">

    <EUSCIBase></EUSCIBase>

    <TXGPIOPort></TXGPIOPort>

    <TXGPIOPin></TXGPIOPin>

    <RXGPIOPort></RXGPIOPort>

    <RXGPIOPin></RXGPIOPin>
</targetCOMM>

To: 

<targetCOMM available="true">

    <EUSCIBase>EUSCI_AX_BASE</EUSCIBase>

    <TXGPIOPort>GPIO_PORT_PX</TXGPIOPort>

    <TXGPIOPin>GPIO_PINX</TXGPIOPin>

    <RXGPIOPort>GPIO_PORT_PX</RXGPIOPort>

    <RXGPIOPin>GPIO_PORT_PX</RXGPIOPin>
</targetCOMM>

IMPORTANT: Make sure you replace GPIO_PORT_PX and GPIO_PINX for the appropriate Port and Pin number in you application. Also verify EUSCI_AX_BASE is updated with the correct EUSCI_A0_BASE or EUSCI_A1_BASE value.
