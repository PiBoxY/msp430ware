Introduction
FlowESI GUI

FlowESI GUI allows users to generate configuration code for the Extended Scan Interface (ESI) module on MSP430 devices via a graphical block diagram tool. The tool currently supports code generation up to 3 LC sensors.

Code Generation

FlowESI GUI allows code generation in two formats:

  • Fully functional projects
  • ESI configuration files

Project Generation is supported in the following IDEs:

  • Code Composer Studio v6 or later
  • IAR Embedded Workbench for MSP430 6.10.2 or later

Target Communication

FlowESI GUI enables target communication on EVM430-FR6989 or custom boards via a HID/UART communication bridge. Enabling target communication allows users to:

  • Get step-by-step instruction to calibration LC sensors.
  • Obtain TSM and DAC calibration values.
  • Monitor rotation counters values.
  • Obtain recalibration information.
  • Obtain ESI debug information.

All of this functionality is enabled without the need of an IDE.
