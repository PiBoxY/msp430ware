Installation and Usage

As part of MSP430Ware installation

Generally FlowESI GUI is installed as part of MSP430ware.

After installing MSP430Ware restart Code Composer Studio. To launch the FlowESI GUI:

  1. Open TI Resource Explorer (View-->TI Resource Explorer). Under packages select "MSP430Ware".



  2. Expand Development Tools and select FlowESI GUI.



  3. Finally, click on "Launch FlowESI GUI".

FlowESI GUI standalone installation

  1. Download FlowESI GUI installer
  2. Run the installer FlowESIGUI_x_xx_xx_xx_setup.exe
  3. Follow installer instructions
  4. To launch tool. Go to Start Menu --> All Programs --> Texas Instruments --> FlowESI GUI --> FlowESI GUI
