Installation and Usage

MSP430 IEC60730 Software Package can be installed as part of MSP430ware Driver Library or it can be use as a stand alone library.

Host Support

This release supports installation and development on Microsoft Windows XP Workstations.

Dependencies

  • MSP430 IEC60730 Software Package Dependencies:
    • CCS5.4+
    • IAR 5.40+

This release supports the following devices:
  • MSP430G23xx
  • MSP430G24xx
  • MSP430G25xx
  • MSP430G2x44
  • MSP430G2x77
  • MSP430F5xx
  • MSP430FR57xx
  • DRV91670

Validation Information

This release is a minor release version and has been tested.

Examples

The examples can be found in MSP430 IEC60730 Software Package Example Projects. The directory contains four examples:

  • IEC60730_emptyProject
  • IEC60730_msp430f5529_example
  • IEC60730_msp430g2553_example
  • IEC60730_msp430fr5739_example

For CCS, the examples can be imported using the import wizard.

For IAR, double click on the .eww file inside the IAR directory of each example project.

Please consult the MSP430 IEC60730 Software Package Users Guide for detailed instructions on how to to run the examples.
