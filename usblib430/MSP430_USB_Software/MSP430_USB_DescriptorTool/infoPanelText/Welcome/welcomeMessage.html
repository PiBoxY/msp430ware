Welcome

Welcome to the MSP430 USB Descriptor Tool! This tool quickly generates customized, reliable descriptor files for use with MSP430's USB API stacks. It's a part of the MSP430 USB Developers Package.

For an overview of how to use this tool, choose Help --> Help or press F1.

For specific help on each individual field, float the mouse over it or click on it; help will appear in this Help Pane.

For detailed reference for the entire solution, please see the USB API Programmer's Guide, located within the USB Developers Package.
