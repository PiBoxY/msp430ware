XT2 Oscillator Frequency

The XT2 frequencies of 4MHz, 8MHz, 12MHz, 16MHz and 24MHz are detected automatically by the USB stack.

Frequencies other than the once listed above need to be manually set (see Programmerâ€™s guide, Programmers_Guide_MSP430_USB_API.pdf for detailed steps)

The API enables the XT2 oscillator when USB is active, and optionally disables it during USB suspend. It uses XT2 as a reference to the USB PLL.

XT2 needs one of these applied to it:

  • a crystal/resonator (minimum of 4MHz)
  • a clock signal from elsewhere on the board, via XT2's "bypass" mode (minimum of 1.5MHz)

Many input frequencies are possible, allowing cost reduction:

  • maybe the clock is available elsewhere on the board, saving the cost of the crystal.
  • maybe another device on the board can use the MSP430's clock, saving the cost of the crystal.
  • a crystal can be chosen based on the lowest cost.
