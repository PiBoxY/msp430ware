Device Release Number

Associated descriptor fields: bcdDevice (in the Device Descriptor)

This value allows the version number of your USB product to be identified on the host. Enter the value here, in the format 0xnnnn, where nnnn is a four-digit hex number.

In Windows, it is can be seen in the Properties for the device, under Firmware Version. Aside from this, it has no serious effect; it's simply for your own use.
